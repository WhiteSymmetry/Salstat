Salstat uses:

  • Python (Salstat is based on Python)
  • wxPython (wxPython is the GUI)
  • Numpy (Numpy does the numbers)
  • SciPy (tests)
  • requests (helps scrape URLs)
  • BeautifulSoup (helps scrape URLs with poorly formed HTML)
  • xlrd (loads Excel format files – and maybe we'll use xlwt in the future)
  • ezodf (loads LibreOffice Calc files)
  • sas7bdat (loads SAS files)
  • JQuery (helps present the output)
  • Twitter Bootstrap (makes the output look much better)
  • HighCharts (does Salstat's charts)
